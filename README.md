# 🎵 音楽制作スタジオ (Kids Beat Maker)

ブラウザを開くだけですぐに使える、子供向け（小学生〜）の直感的な音楽制作アプリ（DAW）です。 専門知識がなくても、マス目を「ポチポチ」と塗るだけで誰でも簡単にオリジナル曲を作ることができます。プログラミング教育やクラブ活動の教材として最適です。

## ✨ 主な機能 (Features)

*   **直感的なステップシーケンサー**:
    
    *   メロディトラック（8音階）とリズムトラック（キック・スネア・ハイハット）を搭載。なぞり入力にも対応。
        
*   **絶対にリズムが狂わないAudio Engine**:
    
    *   商業用DAWと同じ「Lookahead Scheduling」を採用し、ブラウザ上で正確なリズム再生を実現。
        
*   **魔法のサウンド設定**:
    
    *   4種類の音階（長調、短調、琉球、祭囃子）と7種類の楽器音（ピアノ、オルゴール、8bit、ギターなど）をボタン一つで切り替え。
        
    *   空間を彩る「反響（ディレイ）」「残響（リバーブ）」エフェクト搭載。
        
*   **AI一発作曲機能 (AI Composer)**:
    
    *   ChatGPTやGeminiなどの生成AIに読み込ませる「専用プロンプト」を内蔵。AIが生成したJSONコードを読み込むことで、イメージ通りの曲が一瞬で完成します。
        
*   **MP3エクスポート機能**:
    
    *   作成した曲を、指定したループ回数でMP3ファイルとしてダウンロード可能。動画のBGMなどにも使えます。
        
*   **子供に優しい安心設計**:
    
    *   すべての漢字に「ふりがな（ルビ）」付き。
        
    *   初回起動時のインタラクティブな「ガイドツアー」搭載。
        
    *   変更するたびにブラウザに自動保存されるオートセーブ機能。
        
*   **完全レスポンシブ対応**:
    
    *   PC、タブレット、スマートフォンのどの画面サイズでも快適に操作可能。
        

## 🚀 使い方 (Usage)

1.  **音を置く**: 画面右側のマス目をクリック、またはなぞって色を塗ります。
    
2.  **再生する**: 左上の「再生」ボタン（またはスペースキー）を押して音楽を聴いてみましょう。
    
3.  **音を変える**: 左側のパネルから「音階」や「楽器」を変えたり、「やまびこ」のチェックを入れて響きを変えたりできます。
    
4.  **保存する**: 自動保存されていますが、「MP3出力」ボタンを押すと、音声ファイルとしてパソコンに保存できます。
    

### 🤖 AI作曲機能の使い方

1.  左下の「AI作曲」ボタンをクリックします。
    
2.  モーダル内の「AIへの指示（プロンプト）をコピー」ボタンを押します。
    
3.  お好きな生成AI（ChatGPT, Claude, Geminiなど）を開き、コピーした文章を送信します。
    
4.  AIから「どんなイメージの曲を作りたいですか？」と聞かれるので、「宇宙を探検するような曲」「明るくて元気な曲」など自由に答えます。
    
5.  AIが出力した `JSONコード` をコピーし、アプリの読み込み欄に貼り付けて「魔法で曲を生成！」ボタンを押せば完成です。

## 🛠️ 技術スタック (Tech Stack)

*   **Frontend**: React, Tailwind CSS, Lucide React (Icons)
    
*   **Audio**: Web Audio API (Oscillator, BiquadFilter, Convolver, Delay, WaveShaper)
    
*   **Export**: lamejs (MP3 Encoding)
    
*   **Architecture**: 単一ファイルコンポーネント (Single File React App)
    

## 💻 開発者向け (Getting Started)

このアプリケーションは、外部サーバーやデータベースを必要としない完全なスタンドアロン形式で設計されています。

`App.jsx` 1ファイルにすべてのロジックが含まれており、Viteなどのビルドツールを使用して即座にGitHub Pagesへデプロイ可能です。

```
# クローン
git clone [https://github.com/yourusername/kids-beat-maker.git](https://github.com/yourusername/kids-beat-maker.git)

# 依存関係のインストール (lucide-reactが必要です)
npm install

# 開発サーバーの起動
npm run dev
```
